# Decoder-Unip

⚛️ A função deste projeto é criar uma aplicação web simples com uma função de decodificar (e nesse caso, também codificar) cifras e outras
maneiras de criptografia. 

## Este projeto foi construído com

- Angular v19
- Tailwind CSS
- SCSS
- Typescript
- API Rest e GraphQL

### Como rodar o projeto

Para rodar o projeto, primeiramente clone o repostiório na sua pasta preferida com o comando:

```bash
git clone https://github.com/Pedro-B-Siqueira/projeto-aps.git
```

Após isso, você deve entrar na pasta do projeto e utilizar o comando `npm install` para instalar as dependências.

Com as dependências baixadas você pode iniciar o projeto utilizando o comando `npm run start-dev` ou então `npm nun start-prod`.

### Boas práticas na hora de realizar as alterações

Para criar alterações nesse projeto você deve fazer o seguinte:

- 1- Crie uma branch nova:
    [https://streamable.com/1qexmn](Vídeo de tutorial)
- 2- Dê um `git pull origin master --rebase`, faça o que precisa no código e então garanta que está tudo funcionando antes de enviar;
- 3- Adicione os arquivos na fila de mudança com `git add .` ou `git add --all`, verifique se os arquivos que você quer enviar estão em verde no comando `git status`
- 4- Execute o comando `git commit -m "<mensagem-de-commit>` *lembre-se de criar uma mensagem de commit que condiza com o que você mexeu no código, nada de mensagem como: "mexi numas paradas".
- 5- Finzalize com o comando `git push origin <nome-da-sua-branch>`, caso tenha esquecido de mandar alguma coisa, repita o processo acima, porém, ao inves do comando de commit padrão, utilize `git commit --amend` e depois dê um `ctrl + X`. Se nenhum erro surgir, rode o mesmo comando do `git push` mas dessa vez com um `--force` após o nome da sua branch (não esqueça de dar espaço).
